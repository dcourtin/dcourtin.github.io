## À propos de ce repo

Ce CV est aussi disponible en version HTML imprimable (A4) stylée avec Tailwind CSS v4.

```bash
npm install
npm run dev      # watcher Tailwind (assets/app.css)
npm run build    # build minifié pour la prod
```

Ouvrir `index.html` dans un navigateur. Pour exporter en PDF : **Imprimer → Enregistrer en PDF → Marges : aucune**.

Les couleurs et polices sont définies dans [`src/input.css`](src/input.css) sous `@theme`.
